# Life Briefing

自动生活简报系统 - 追踪每日工作、思考、健康与成长

## 🎯 核心功能

- **每日简报**: 自动生成结构化日报
- **生活仪表盘**: 可视化追踪关键指标
- **原则检查**: 自动验证三条核心原则执行情况

## 🚀 快速开始

访问仪表盘: https://zhongyi-byte.github.io/life-briefing/dashboard/
