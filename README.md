# Life Briefing

自动生活简报系统 - 追踪每日工作、思考、健康与成长

## 🎯 核心功能

- **每日简报**: 自动生成结构化日报
- **生活仪表盘**: 可视化追踪关键指标
- **原则检查**: 自动验证三条核心原则执行情况

## 🚀 快速开始

访问仪表盘: https://zhongyi-byte.github.io/life-briefing/dashboard/

## 📊 追踪指标

| 指标 | 目标 |
|------|------|
| 睡眠时长 | 7-8h |
| 运动时长 | ≥30min |
| 深度思考 | ≥500字 |
| 公开输出 | ≥1次/周 |
| 原则遵循 | 100% |

## 🔥 发展方向

- GitHub 风格热力图：从 daily/weekly 简报加载真实数据，点击即可查看详情
- 自动周报：基于一周的日简报自动生成周度回顾与改进建议
- 数据来源扩展：接入可穿戴设备与日常应用数据，减少人工录入
- 反馈闭环：在仪表盘上提供小目标与提醒（如情绪/睡眠告警）
